## Table of Contents

- [Usage](#usage)

- [Build](#build)

## Usage

Add this plugin to an Nx workspace:

```
yarn add @plugified/nx-fastify
```

or

```
npm install @plugified/nx-fastify --save
```

## Project schematics

Generate your project:

```
nx g @plugified/nx-fastify:application my-app
```

## Build

Build your project:

```
nx build my-app
```

## Serve

Serve with:

```
nx serve my-app
```
